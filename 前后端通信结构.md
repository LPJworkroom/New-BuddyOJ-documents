### 后端响应：

```
{
	code,
	msg,
	data:{
		data,
		totalNumber
	},
}
```

code取值：

	- success
	- fail
	- need_refresh_token
	- need_login

msg: 后端直接告诉用户错误原因


token:
```
{
	userId,
	tokenType:'token',
	role,
	teamRole:[
        {
            teamId,
            role
        },
	],
	expiresAt,
}
```

refresh token:

```
{
	userId,
	tokenType:'refresh',
	expiresAt,
}
```
